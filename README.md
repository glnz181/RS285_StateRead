# RS485 State Read

## Proje Tanımı
Bu proje, RS485 haberleşme altyapısına sahip bir indikatör cihazdan veri okunması ve temel komutların gönderilmesi amacıyla geliştirilmiştir.

## Amaç
RS485 haberleşme yapısını anlamak ve Modbus/standart iletişim yöntemleri ile cihazdan veri alıp komut göndermeyi gerçekleştirmektir.

## Kullanım Amacı
- Cihazdan veri okuma (state/weight bilgisi)
- Komut gönderme (dara, reset vb.)
- Haberleşme yapısının test edilmesi

## Kullanılan Teknolojiler
- Python (seri haberleşme)
- RS485 haberleşme protokolü

## 📊 Sonuç
RS485 üzerinden veri okuma işlemi başarıyla gerçekleştirilmiş, sistemin çalışma mantığı anlaşılmış ve gömülü sistem tarafına aktarım süreci sağlanmıştır.
